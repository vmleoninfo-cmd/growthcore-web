## 1. VOCABULARIO CANÓNICO

### Verbos del sistema — los que Growth Core usa como propios

| Verbo | Por qué pertenece al sistema |
|---|---|
| **Detectar** | Verbo central — deriva del Pilar 1. Growth Core detecta antes de actuar |
| **Identificar** | Sinónimo preciso de detectar para variación — sin perder especificidad |
| **Alertar** | Acción del sistema hacia el emprendedor — activa, no pasiva |
| **Priorizar** | Pilar 4 — el sistema ordena qué atender primero |
| **Calificar** | Aplicado a leads — diferencia el agente de WhatsApp de un bot |
| **Registrar** | El sistema captura datos de manera automática y estructurada |
| **Instalar** | Pilar 2 — Growth Core instala, no el cliente configura |
| **Conectar** | Pilar 3 — las herramientas se conectan entre sí |
| **Operar** | El sistema opera en el negocio — no "funciona", opera |
| **Integrar** | Pilar 3 — datos y herramientas integradas |
| **Entrenar** | Específico para el Agente WhatsApp — el agente se entrena con el negocio del cliente |

### Sustantivos del sistema — los que definen la identidad del ecosistema

| Sustantivo | Uso correcto |
|---|---|
| **Sistema operativo comercial** | Descriptor central de qué es Growth Core como conjunto |
| **Cerebro operativo** | Metáfora canónica del ADN — solo cuando se quiere transmitir la idea de inteligencia aplicada |
| **Ecosistema** | Cómo se nombra el conjunto de las 4 herramientas — no "plataforma" ni "suite" |
| **Infraestructura** | Describe lo que Growth Core instala en el negocio del cliente |
| **Diagnóstico** | El proceso de identificar el problema antes de cualquier solución |
| **Acción concreta** | El output del sistema — siempre concreta, nunca abstracta |
| **Acción prioritaria** | Cuando hay múltiples acciones posibles, el sistema prioriza una |
| **Cuello de botella** | El problema específico que más bloquea el crecimiento del negocio |
| **Score** | Válido cuando se explica su escala y lo que activa — no como métrica sola |
| **Flujo de ventas** | Preferible a "pipeline" para el segmento objetivo |
| **Oportunidad perdida** | Describe los leads, clientes o presupuesto que se pierde sin sistema |
| **Onboarding** | El proceso de instalación e implementación — no "configuración" ni "setup" al hablar con el cliente |

### Adjetivos del sistema — los que describen el ecosistema y sus outputs

| Adjetivo | Uso correcto |
|---|---|
| **Específico/a** | Toda solución de Growth Core es específica — no genérica |
| **Concreto/a** | Toda acción entregada es concreta — no abstracta |
| **Instalado/a** | Las herramientas se entregan instaladas — no "disponibles" ni "accesibles" |
| **Conectado/a** | Las herramientas y los datos del cliente están conectados |
| **Integrado/a** | El ecosistema funciona integrado — no como herramientas aisladas |
| **Entrenado/a** | El Agente WhatsApp se entrega entrenado para el negocio |
| **Accionable** | Los datos del sistema producen acciones — no reportes para contemplar |

## 2. FRASES FUNDACIONALES

Estas frases describen Growth Core en sus diferentes niveles. Se pueden usar directamente o como base de variaciones. No se reemplazan por sinónimos genéricos.

### Cómo describir qué ES Growth Core

> "Growth Core es el sistema operativo de tu negocio."

> "Un cerebro que detecta dónde se pierden oportunidades y te dice qué hacer."

> "Infraestructura comercial instalada en tu negocio — sin que toques código."

> "El sistema que conecta tus campañas, leads y clientes en un solo lugar con inteligencia."

### Cómo describir qué HACE Growth Core

> "Detecta qué está fallando antes de que afecte tus resultados."

> "Entrega la acción concreta que resuelve cada problema que detecta."

> "Responde leads en segundos mientras tú atiendes otro cliente."

> "Sabe cuándo y cómo contactar cada cliente para cerrar más ventas."

### Cómo describir PARA QUIÉN es Growth Core

> "Para emprendedores que ya tienen clientes y campañas, y sienten que están dejando dinero sobre la mesa."

> "Para negocios que no tienen equipo técnico pero necesitan que el sistema funcione igual."

> "Para quienes ya invierten en publicidad y necesitan saber si esa inversión está trabajando."

### Cómo describir la DIFERENCIA de Growth Core

> "No es un CRM que configuras — es un sistema que opera en tu negocio desde el primer día."

> "No gestionamos tus campañas — te damos el sistema para que tus campañas trabajen mejor solas."

> "No es un bot de respuestas — es un agente entrenado con el guion de ventas de tu negocio."

## 4. CÓMO HABLAR DEL CLIENTE

### Formas correctas de referirse al cliente objetivo

| Forma | Uso |
|---|---|
| **"Tu negocio"** | Forma primaria — crea proximidad y especificidad |
| **"Emprendedor"** | Para contenido de captación donde no hay relación previa |
| **"Dueño del negocio"** | Para contextos donde se quiere mostrar autonomía y responsabilidad del cliente |
| **"Tu equipo"** | Cuando la empresa tiene más de 1 persona y se quiere hablar de adopción |

### Formas prohibidas de referirse al cliente

| Prohibido | Por qué | Alternativa |
|---|---|---|
| "Pequeña empresa" | Suena diminutivo — el cliente no se ve a sí mismo como pequeño | "Negocio de servicios" / "tu negocio" |
| "Usuario" | Lenguaje de software-as-a-service genérico — no de sistema instalado | "Tú" / "tu equipo" |
| "Cliente final" | Jerga interna que no debe llegar a la comunicación externa | "El cliente de tu negocio" |
| "PYME" | Etiqueta burocrática — no genera identificación | "Negocio con equipo pequeño" |

## 5. CÓMO HABLAR DE LOS RESULTADOS

### Regla central: resultado anclado al problema

No se comunica resultado sin el problema que lo genera. El resultado no es una promesa — es la consecuencia observable de detectar y resolver un problema específico.

**Estructura correcta:**
> "Smart Ads detectó que [herramienta X] estaba desperdiciando [porcentaje] del presupuesto en clics de baja calidad. Al redirigir ese presupuesto, los leads calificados de MOVE aumentaron [número] en [tiempo]."

**Estructura prohibida:**
> "Con Growth Core, nuestros clientes aumentan sus ventas en más del 50%."

### Cómo citar resultados sin prometer

- Con cliente real nombrado: "MOVE — los datos de monitoreo mostraron X en Y semanas"
- Sin cliente nombrado: "En negocios con este perfil, el sistema detecta [problema] en los primeros [tiempo]"
- Nunca: porcentajes de mejora sin fuente, contexto o condición

### Vocabulario de resultados por tipo

| Tipo de resultado | Vocabulario canónico | Vocabulario prohibido |
|---|---|---|
| Leads | "leads calificados" · "leads respondidos a tiempo" · "leads registrados" | "más leads" · "leads explosivos" · "leads ilimitados" |
| Campañas | "presupuesto recuperado" · "alerta antes de pérdida" · "score optimizado" | "campañas exitosas" · "ROI astronómico" · "campañas que convierten" |
| Clientes | "clientes priorizados" · "seguimiento sistemático" · "cierre identificado" | "más clientes" · "clientes felices" · "retención perfecta" |
| Tiempo | "mientras tú atiendes otro cliente" · "en los primeros 30 segundos" · "antes de que afecte resultados" | "ahorra tiempo infinito" · "trabaja menos" · "automatización total" |
| Dinero | "presupuesto no desperdiciado" · "inversión que trabaja" | "ingresos pasivos" · "gana más dinero" · "riqueza automatizada" |

## 6. PALABRAS Y FRASES PROHIBIDAS

### Nivel 1 — Prohibición absoluta (nunca en ningún contexto)

| Prohibido | Razón | Alternativa |
|---|---|---|
| "Ingresos pasivos" | Lenguaje de promesa vacía — representa exactamente lo que Growth Core no es | "Tu sistema opera mientras tú estás en otra parte" |
| "Desde la comodidad de tu hogar" | Aspiración personal del fundador proyectada al cliente | "Tu agente responde leads aunque estés en otra reunión" |
| "Trabaja menos, gana más" | Promesa que destruye credibilidad con el público objetivo | Describe el problema específico que se elimina |
| "Escala ilimitadamente" | Promesa sin condiciones ni mecanismo | "Crece el volumen de leads que puedes atender sin agregar personal" |
| "Revolución" / "Disruptivo" / "Game changer" | Hype sin sustancia — señal de que no hay argumento real | Describe qué hace diferente, específicamente |
| "Sin esfuerzo" / "Sin trabajo" | Irreal y contradice la implementación real | "Sin que toques código" / "sin aprender un software nuevo" |
| "Resultados garantizados" | Promesa incontrolable — crea expectativas sin base | "En negocios con este perfil, el sistema detecta X en Y tiempo" |

### Nivel 2 — Prohibición contextual (no como argumento central)

| Prohibido como argumento central | Cuándo sí puede usarse | Cómo |
|---|---|---|
| "Personalizado" | Solo como descriptor secundario, nunca como diferenciador | "CRM configurado específicamente para tu tipo de negocio" |
| "Optimizar" | Solo cuando se especifica qué y cómo | "Smart Ads optimiza la distribución del presupuesto entre creativos cuando detecta CTR en caída" |
| "Plataforma" | Solo para referirse a Meta, WhatsApp, etc. — nunca para Growth Core | "Growth Core se conecta con la plataforma de Meta Ads" |

> **Nota de coherencia con CAT.3:** Las métricas (CTR, CPL, ROAS) acompañadas de mecanismo explicado y acción concreta son correctas — corresponden al Argumento 2 de CAT.3 (mecanismo explicado). Esta prohibición aplica únicamente cuando se usan como argumento central sin contexto accionable ("Aumenta tu CTR" sin explicar cómo ni qué activa eso).
| "Automatizar" como promesa | Solo como descriptor de función específica | "El agente automatiza la respuesta inicial en WhatsApp con el guion de tu negocio" |
| "Potenciar" | Solo si va seguido de qué se potencia y cómo | Si no hay especificidad, reemplazar por el verbo de acción |

### Nivel 3 — Vocabulario de agencia prohibido para Growth Core

Estas palabras son correctas en otros contextos pero no pertenecen al ADN de Growth Core:

- "Estrategia de marketing" — Growth Core no es estrategia, es sistema
- "Branding" — no es servicio de Growth Core
- "Community management" — no es servicio de Growth Core
- "Funnel" — usar "flujo de ventas" o "proceso de captación"
- "Lead magnet" — usar "contenido de captación" o "primera conversación"
- "Onboarding" en comunicación externa al cliente — usar "instalación" o "proceso de implementación"

## 7. REGISTRO DE ESCRITURA

### Estructura de oración preferida

- Oraciones cortas. Una idea por oración.
- Sujeto activo: el sistema hace, detecta, entrega — no "se puede hacer", "se detecta"
- Verbo al inicio cuando hay instrucción: "Detecta. Actúa. Crece." — no "Tu negocio podría crecer si detectas y actúas"
- Sin gerundios acumulados: no "estando conectado y funcionando y detectando"

### Longitud por tipo de pieza

| Pieza | Longitud ideal |
|---|---|
| Post de redes (texto) | 80-150 palabras — un problema, un mecanismo, una acción |
| Título o hook | 10-15 palabras — uno solo punto, preciso |
| Descripción de herramienta | 2-3 oraciones — qué detecta + qué entrega + en qué tiempo |
| Email de informe (Smart Ads) | 150-250 palabras — score + alertas + acciones |
| Propuesta a cliente | Sin límite — pero cada sección en no más de un párrafo |

### Puntuación y formato

- Guiones (—) para clarificar, no para decorar
- Negritas para el concepto central de cada párrafo — no para decoración
- Listas solo cuando hay 3 o más elementos paralelos — no para fragmentar ideas que fluyen en prosa
- Mayúsculas en nombres propios del sistema: Smart Ads Dashboard, Agente WhatsApp con IA, CRM con IA
